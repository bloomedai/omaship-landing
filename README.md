## ğŸ“Š Deploy to GitHub Pages

### Automatic Deployment (Recommended)
1. Fork or clone this repository
2. Enable GitHub Actions in your repository settings
3. Push changes to the main branch
4. Your site will be automatically deployed to `https://yourusername.github.io/omaship-landing`

### Manual Setup
1. Go to your repository settings
2. Scroll to "Pages" section
3. Select "Deploy from a branch"
4. Choose "main" branch and "/" (root) folder
5. Save and wait a few minutes
6. Your site will be live at `https://yourusername.github.io/omaship-landing`

### Custom Domain
1. Add a `CNAME` file to your repository root with your domain name
2. Configure DNS settings with your domain provider
3. Enable "Enforce HTTPS" in GitHub Pages settings

For detailed instructions, see our [Deploy Guide](https://docs.github.com/en/pages/getting-started-with-github-pages).
